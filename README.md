# Alumni Management System

This is a web-based Alumni Management System developed using PHP and MySQL.

## 📌 Features
- Alumni Dashboard
- Chat System
- Discussion Forum
- Job Posting System
- Role-based Login (Admin, Alumni, Student)

## 🛠 Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL

## ⚙️ How to Run

1. Install XAMPP
2. Move project folder to htdocs
3. Start Apache & MySQL
4. Open phpMyAdmin and create database
5. Import SQL file
6. Run project on localhost
